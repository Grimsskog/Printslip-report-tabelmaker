Simple HTML document for displaying a Prinslip-report XML file from "Pickup From Shelf" in Alma for easy prints. 
The document is intended to be a quick and easy way to print a list of requests to be collected. Download the print slip report as an XML file in Alma and upload it to the HTML document to generate the list.

Two versions: <br>
Norwegian: printslip_report_fremviser.html <br>
English: printslip_report_tabelmaker.html
